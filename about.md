This tool lets you explore changes in the U.S. native‑born and foreign‑born populations using the American Community Survey (ACS) 1‑year estimates from 2005–2024. You can view trends for the nation, all states, and all counties and places (i.e., cities) with populations of 65,000 or more. 

ACS data for 2025 is expected to be published in September 2026. Going forward, this tool can help people track how the foreign‑born population shifts as the Trump administration’s immigration enforcement efforts unfold.

This app was built by Ari Lamstein. If you have questions or feedback about it, you can reach me through my [website](https://arilamstein.com/).

The app is built with Streamlit and powered by the `acs‑nativity` Python package.
